# ASP.NET Core & Angular - Full Stack Application

https://github.com/user-attachments/assets/f51f62ce-c5e6-4afb-943a-c0e9f4451b5c

This project was developed as part of an advanced ASP.NET Core and Angular course focused on building a complete real-world web application from scratch using modern development practices and technologies.

Throughout the course, I worked with:

* ASP.NET Core (.NET 9)
* Angular 20
* Entity Framework Core
* JWT Authentication
* AutoMapper
* TailwindCSS & DaisyUI
* Azure Deployment
* Reactive Forms
* Filtering, Sorting & Pagination
* Private Messaging System
* Drag & Drop Photo Upload
* SignalR for real-time communication

One of the most interesting parts of the project was learning and implementing SignalR to handle real-time user presence and live messaging between users, providing instant updates and a more dynamic user experience.

The course also reinforced best practices for structuring scalable full stack applications, API error handling, authentication flows, and frontend/backend integration using modern Angular and ASP.NET Core architectures.
